# Projet 2 – Analyse de données de systèmes éducatifs

## Contexte

Une entreprise souhaite se développer à l’international.  
L’objectif est d’identifier les pays les plus pertinents à partir de données éducatives.

## Objectif

Réaliser une analyse de données afin de formuler des recommandations stratégiques.

## Données

- Données de la Banque mondiale (éducation)
- Indicateurs multi-pays et multi-années

## Méthodologie

- Nettoyage des données
- Sélection des indicateurs pertinents
- Analyse univariée et bivariée
- Visualisation des résultats

## Outils utilisés

- Python
- Pandas / NumPy
- Matplotlib / Seaborn
- Jupyter Notebook

## Résultats

- Identification de pays cibles
- Mise en évidence de tendances éducatives
- Recommandations stratégiques

## Compétences développées

- Manipulation de données
- Analyse statistique
- Data visualisation
