# Data Science Projects

Ce dépôt regroupe les projets réalisés dans le cadre de ma formation de Data Scientist (OpenClassrooms).

Ces projets couvrent l’ensemble d’un cycle data :
- préparation et nettoyage des données
- analyse exploratoire
- modélisation statistique et machine learning
- restitution des résultats

## Projets

### Projet 1 – Analyse de données de systèmes éducatifs
Analyse de données internationales afin de produire des recommandations stratégiques d’expansion.

### Projet 2 – Préparation de données pour un organisme de santé publique
Nettoyage, exploration et structuration de données nutritionnelles pour une future application.

### Projet 3 – Prédiction de la consommation énergétique de bâtiments
Modélisation de la consommation énergétique dans une optique de transition écologique.

## Compétences développées

- Python (Pandas, NumPy, Matplotlib, Seaborn)
- Analyse statistique
- Data cleaning
- Machine Learning
- Data visualisation

## Objectif

Présenter des projets concrets basés sur des cas réels, en lien avec des problématiques métier.
